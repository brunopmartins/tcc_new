Na etapa de revisão da literatura foram levantados estudos que abordam modelos computacionais de classificação, reconhecimento facial e a verificação de parentesco a partir das características faciais de um par ou grupo de pessoas.

 Na segunda etapa, foi selecionada a base de dados Families in the Wild, com aproximadamente 1,000 famílias, 10,676 pessoas diferentes, 30,725 faces nas fotos\cite{robinson2018visual}.
 
 O modelo proposto para o trabalho foi uma implementação em python de uma rede neural convolucional siamesa do VGGFace com o  RESNET. Levando em consideração a proximidade das tarefas de verificação facial e verificação de parentesco, o modelo é prétreinado usando a base de dados Labeled Faces in the Wild\cite{huang:inria-00321923}, após isso são removidas as camadas do topo e são removidas.Para a utilização do conjunto de dados anteriormente selecionada para esse trabalho, Families in the Wild, foi realizado um pré-processamento das imagens utilizando a técnica de afunilamento\cite{huang2007unsupervised}. Realizado o pré-processamento da base, as imagens resultantes são utilizadas para treinar o modelo, e o modelo é testado utilizando cross-validation, demonstrando resultados de aproximadamente 79\% de acurácia.
