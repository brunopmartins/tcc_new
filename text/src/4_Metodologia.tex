\chapter[Metodologia]{Metodologia}
\label{cap:Metodologia}

O objetivo deste trabalho é o desenvolvimento de um modelo de classificação de parentesco entre indivíduos usando imagens de suas faces . A metodologia proposta para alcançar tal objetivo foi composta pelas seguintes etapas:
\newline
\begin{enumerate}
    \item Revisão da literatura sobre modelos de classificação.
    \item Seleção das bases de dados.
    \item Proposta do modelo de classificação.
    \item Experimentos com o modelo sobre a base de dados.
    \item Avaliação do modelo proposto quanto a seus objetivos.
\end{enumerate}
    
    A primeira etapa da metodologia foi a revisão da literatura. Nesta fase, foram
levantados os estudos que se relacionam à proposta deste trabalho. Como resultado, foram
gerados: (1) o capítulo de Trabalhos Relacionados.

    A segunda etapa se refere à seleção da base de dados. Nesta etapa, foram estudadas as basee de dados referentes à classificação da relação de paternidade entre dois indivíduos através de imagens da face de cada um. Desse modo foi possível identificar os conjuntos de daos relevantes, possíveis preprocessamentos das imagens das bases para melhorar a acurácia do modelo e selecionar métodos de classificação adequados para o banco de dados apresentado.
    
    A terceira etapa da metodologia consiste na proposta de um modelo de classificação para o problema da avaliação relação de paternidade. Nesta fase, foi realizada uma avaliação dos métodos de classificação pertinentes ao problema e levantadas técnicas que poderiam influênciar na classificação.
    
    Na etapa seguinte, foram realizados experimentos com o modelo de classificação proposto sobre a base de dados. Como resultado, foram gerados dados relativos à eficácia dos métodos propostos.

    Emfim, na quinta e última etapa, há a avaliação dos resultados proveniente dos experimentos da fase anterior. Com essa avaliação, foi possível comparar o modelo  proposto com os modelos usados na competição "Northeastern SMILE Lab - Recognizing Faces in the Wild" do Kaggle.
