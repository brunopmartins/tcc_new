\chapter[Conclusões Preeliminares]{Conclusões Preeliminares}
\label{cap:Conclusões Preeliminares}

Este trabalho propõe a construção de um modelo computacional de classificação para a verificação visual de parentesco automatica a partir de fotos da face de duas pessoas.

Até o momento foram definidos a metodologia para o desenvolvimento do classificador, um cronograma para a realização do projeto e foram realizadas as seguintes etapas propostas: (1) Revisão da Literatura, (2) Estudo do Banco de Dados, (3) Proposta de Modelo. Está ocorrendo atualmente a etapa: (4) Execução dos Experimentos.

O trabalho realizado até o momento mostra a viabilidade prática da avaliação do novo método. Portanto, a continuação do trabalho é justificada, já que está sendo realizado dentro do cronograma definido.
