+ relation-aux 0.05 + role-matched hard negatives 30%) with the AdaFace backbone

memorise, but the safeguard (decline-based) never tripped, so it was stopped manually.

## Verdict

**LoRA underperforms the full stage-4 unfreeze** — AUC 0.8700 is ~0.006–0.011
below the R011/R012 unfreeze line, and below on every metric. The hypothesis
(low-rank regularisation raises the generalisable ceiling) did not pan out: LoRA's
val stayed stably near-peak (it didn't memorise), but its ~0.9 M backbone adapters
have less adaptation capacity than the 31 M unfreeze, so it caps lower on test.
Single-run (best ep5); CV would firm it up but the direction is unfavourable.
Useful negative result for the ceiling story.
