## How to Resume

```bash
cd models/02_vit_facor_crossattn

SKIP_INSTALL=1 TRAIN_DATASET=fiw EPOCHS=50 PATIENCE=25 \
FREEZE_VIT=0 LOSS=contrastive MARGIN=0.3 TEMPERATURE=0.3 \
NEGATIVE_RATIO=2.0 TRAIN_NEGATIVE_STRATEGY=relation_matched \
EVAL_NEGATIVE_RATIO=1.0 EVAL_NEGATIVE_STRATEGY=random \
LEARNING_RATE=2e-6 WEIGHT_DECAY=5e-5 SCHEDULER=cosine \
WARMUP_EPOCHS=8 CROSS_ATTN_LAYERS=2 CROSS_ATTN_HEADS=8 \
DROPOUT=0.25 RESUME=output/032/checkpoints/best.pt \
bash AMD/run_pipeline.sh
```

**Note:** Resume will create a new run ID (e.g., 033). The model weights and optimizer state will be restored from the checkpoint, but the epoch counter and scheduler state depend on what's saved in the checkpoint. Verify the starting LR matches expected cosine decay from epoch 8.
