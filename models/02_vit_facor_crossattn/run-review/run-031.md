## Analysis

### Improvement over Run 030
- **AUC**: 0.822 → **0.850** (+0.028)
- **F1**: 0.751 → **0.779** (+0.028)
- **Recall**: 82.4% → **94.1%** (+11.7%)
- No-freeze + lower LR clearly better than freeze-then-unfreeze for FIW

### Persistent Issues
1. **Early peak then overfit** — best at epoch 3, still during warmup phase. The model memorizes quickly and stops generalizing.
2. **Low non-kin specificity** — 56.4% non-kin accuracy means 43.6% false positive rate. The threshold (0.900) is very aggressive.
3. **Val/test gap** — AUC 0.881 vs 0.850, suggesting mild overfitting.

### Key Insight
The model peaks during warmup (epoch 3 at LR=3e-6) — the optimal LR might be even lower. Once LR rises above ~3e-6, the model starts overfitting. This suggests trying:
- **Even lower peak LR** (2e-6) with longer warmup
- **Stronger weight decay** to slow memorization
- **More data augmentation** to regularize
